# GIFs Directory

This directory contains animated GIFs used in the GitHub profile README.

## Usage

All GIFs are referenced using GitHub's raw content URLs:
```
https://github.com/AyushGupta0202/AyushGupta0202/blob/main/gifs/FILENAME.gif?raw=true
```

## GIF Sources

All GIFs are from open-source repositories:
- https://github.com/Anmol-Baranwal/Cool-GIFs-For-GitHub
- https://readme-workflows.github.io/Readme-Icons/

## Adding GIFs

1. Upload GIF files to this directory
2. Reference them in README.md using the raw URL format
3. Ensure GIFs are optimized for web (under 1MB recommended)
